# Validate Hex

A simple coding challange designed to validate user input via regular expression.

## Run Locally

Clone the project

```bash
  git clone https://github.com/CStanton-dev/validate-hex.git
```

Go to the project directory

```bash
  cd validate-hex
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
